# Character-Preserving Pose Refinement in Generative AI Image Generation Pipelines

An interactive pipeline for adjusting pose and composition in AI-generated character images while preserving the character's appearance, facial impression, and style.

This repository is structured for GitHub Pages. The entry point is `index.html`.
